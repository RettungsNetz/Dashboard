Firebase Online-Modus einrichten
Kostenlos · 24/7 · Keine Server · Einmalige Einrichtung
1
Firebase-Projekt erstellen (kostenlos)
Öffne die Firebase Console und erstelle ein neues Projekt. Keine Kreditkarte nötig — das Spark-Free-Tier reicht völlig aus.
→ console.firebase.google.com öffnen
1. „Projekt hinzufügen" klicken 2. Projektname eingeben (z.B. rettungsnetz-manv) 3. Google Analytics: Nein → Projekt erstellen
2
Realtime Database erstellen
Im linken Menü: Build → Realtime Database → Datenbank erstellen
Region: europe-west1 (Frankfurt) → Im Testmodus starten
Danach Regeln dauerhaft offen setzen: Realtime Database → Regeln → Folgendes einfügen & veröffentlichen:
{ "rules": { ".read": true, ".write": true } }
⚠ Nur für private/fiktive Szenarien — keine echten Patientendaten!
3
Database URL kopieren
Realtime Database → oben die URL kopieren. Sie endet auf .firebasedatabase.app
Beispiel: https://mein-projekt-default-rtdb.europe-west1.firebasedatabase.app
4
URL eingeben & verbinden
URL hier einfügen — alle Geräte im Einsatz-Team müssen dieselbe URL eingeben, um dieselbe Session zu sehen.
